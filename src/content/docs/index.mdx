## Why Astra?

<CardGrid stagger>
	<Card title="Fast and optimized" icon="seti:css">
		With esbuild and postject you can compile your js code to exe in seconds.
	</Card>
	<Card title="ESM and CJS support" icon="node">
		Unlike pkg, Astra supports both ESM (partially) and CJS modules.
	</Card>
	<Card title="Make exe your own" icon="pencil">
		Astra natively supports customizing the exe icon and metadata (version, copyright etc.).
	</Card>
	<Card title="Offline mode" icon="close">
		Working in plane? No problem! Astra can compile your code without internet.
	</Card>
	<Card title="Modern CLI" icon="sun">
		A modern command-line interface designed for speed, clarity, and developers experience.
	</Card>
	<Card title="Written in TypeScript" icon="seti:typescript">
		For developer who want to contribute in Astra project.
	</Card>
